FAQ
Know more about the project, security, how it works and more.

What's that project?
This web application lets you scan your EU Digital COVID certificate (vaccine, test, or healing) and generate a passbook from it. You can then import it into your iPhone's wallet, or if you're using an Android phone, import it into an app that supports it.
How does it work?
When you scan the QR-code, we decrypt it, get the data it contains and create the passbook (the file needed to have a card in your wallet). When it's ready, we compute a digest of your files and send it for digital signature with a certificate on our server without sending your data itself. The server sends the signature back to your phone, and your phone generates the final file and let you save it.
Is it secure?
Well, we put a lot of efforts into making it the most secure we can. Your phone processes all the personal data and only send a "hash" to our server. It's like a verification number based on the files, and it's impossible to use it to know the original content. The server generates a signature for this hash, using Apple's certificate. No passbook can be opened into your wallet without being signed like this. It's another safety from Apple. Then the last step is made on your phone. We let you preview your information before saving the actual pass.
Can I see the code?
Of course. You can read all the source code and dig more into how it works on the Github page of the project. You can even contribute if you want!
I can't scan my certificate.

Try to have a good light source and center sur QR code in the scan zone. The size of the visual aid is the size of the code we require. Also, your code need to be flat (curled paper will not work well)

We support the EU Digital COVID certificate of vaccination, recovery, test. The ones from Switzerland and the United Kingdom are also supported if they are following the EU format.

First, please be sure to use the EU version of your certificate (if you are in Luxembourg, you can download it from myguichet. If you're in France, you need to download it from Ameli's dedicated website).

Ultimately, you can contact us directly or open an issue.

How to save the certificate on Android?

On Android, you can save the .pkpass file and open it with an application that supports this format. Here are some:

  • Stocard (Google Play Store)
  • PassAndroid (F-Droid)
  • WalletPasses (Google Play Store)

You can also screenshot the pass preview we display before download; it contains all the main pieces of information you can find on the cover of your paper certificate.
