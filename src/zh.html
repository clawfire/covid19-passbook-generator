我無法掃描我的憑證。

  1. 擁有良好的光源，並將 QR Code 與掃描區域邊緣對齊。 畫面上顯示的框框大小，是我們需要掃描的 QR Code 代碼的大小。 你的代碼必須是平整的（皺掉的紙本可能會失敗）。
  2. 如果你有 QR Code 的 截圖或圖片，你可以使用「匯入圖片」按鈕。
  3. 請確認你使用正確的歐盟格式證明。 如果不知道如何獲得證明，請參考下列問題。
  4. 我們支援歐盟數位新冠病毒健康證明（疫苗施打、檢測以及治療）。 來自瑞士和英國的證明，若使用歐盟的格式，此網站也支援。
  5. 如果仍然失敗，你可以聯絡我們，或 建立提問。

這專案是什麼？

這個網頁應用程式掃描你的歐盟數位新冠病毒健康證明（疫苗施打、檢測以及治療）並從中產生一張數位票卡。 然後你可以將其匯入到 iPhone 的錢包，若你使用 Android，則可以使用任何支援的應用程式。

我可以將證明儲存到 Android 上嗎？

可以！ 您可以儲存 .pkpass 檔，並使用支援此格式的應用程式來開啟它，例如：

  • Stocard (Google Play Store)
  • PassAndroid (F-Droid)
  • WalletPasses (Google Play Store)

在下載數位證明前，你也可以將我們提供的預覽畫面截圖下來；它包含您在實體憑證封面上看到的所有相關資訊。

這是如何運作的？

當掃描 QR Code 時，我們會讀取資料並建立數位證明（在你的手機錢包中使用的檔案格式）。 我們會傳送一組數位簽章檢查碼到我們的伺服器，但不會儲存你的資料。 在你的手機上會產生出最終檔案，讓你可以儲存。

這安全嗎？

我們花了很多功夫盡力讓它安全。 你的個人資料永遠不會離開你的手機。 Apple 需要數位簽章才能建立數位憑證。 為了產生此簽章，這個網頁只會將你的資料的檢查碼傳送給我們的伺服器，而我們也無法猜測原始內容。 最後一步發生在你的手機上：在儲存票卡之前，先預覽你的票卡資訊。 我們不追蹤任何人，我們不會收集任何個人資訊，我們也不使用 cookies。

我可以看程式碼嗎？

當然可以。 在 GitHub 本專案頁面上，你可以閱讀程式碼並且了解其如何運作。 如果有意願的話，你也可以參與貢獻！
